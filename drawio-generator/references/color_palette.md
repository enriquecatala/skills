| ------------------------------ | ---------- | ------------ | ------------------------ |

| **Backend / API**              | `#d5e8d4`  | `#82b366`    | REST API, GraphQL, gRPC  |
| **Database / Storage**         | `#ffe6cc`  | `#d79b00`    | PostgreSQL, S3, Redis    |
| **Security / Auth**            | `#f8cecc`  | `#b85450`    | OAuth, IAM, Firewall     |
| **Cloud / Network**            | `#e1d5e7`  | `#9673a6`    | VPC, Subnet, CDN         |
| **External / 3rd Party**       | `#f5f5f5`  | `#666666`    | SaaS, Payment gateway    |
| **Messaging / Queue**          | `#fff2cc`  | `#d6b656`    | Kafka, RabbitMQ, SQS     |
| **Monitoring / Observability** | `#d5e8d4`  | `#56a756`    | Prometheus, Grafana      |
| **Container / Group**          | `#f5f5f5`  | `#999999`    | VPC boundary, namespace  |

## Style Modifiers

- `shadow=1` — adds depth
- `rounded=1` — modern rounded corners
- `fontSize=12` — readable labels
- `fontStyle=1` — bold text
- `strokeWidth=2` — visible borders
- `opacity=80` — subtle backgrounds for groups
